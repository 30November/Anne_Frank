## Description
**Anne_Frank** is a repository that aims to facilitate sharing feelings with someone who will not judge. It provides solace for those situations where emotions and thoughts are difficult to share with others.  

## Repository URL
[30November/Anne_Frank](https://github.com/30November/Anne_Frank)

## Features
- **Confidential Sharing**: Designed to allow sharing of non-shareable emotions without judgment.
- Built using **HTML**, **Bootstrap**, **Python**, **Flask**, and **SQLite** for seamless frontend and backend integration.
- Publicly accessible repository.
- Compatible with modern browsers.

## How to Run the Repository

1. **Install Dependencies**: Ensure that Python is installed on your machine along with any required libraries specified in the `requirements.txt` file.
2. **Setup Environment**: Create a virtual environment and activate it:
    ```bash
    python -m venv myenv
    source myenv/bin/activate  # For Linux/Mac
    myenv\Scripts\activate     # For Windows
    ```
3. **Install Python Packages**:
    ```bash
    pip install -r requirements.txt
    ```
4. **Run the Repository Backend**: Start the Python backend server. Typically, run:
    ```bash
    python app.py
    ```
5. **Access Frontend**: Open the HTML files in your browser or use one of the tools mentioned below to serve the frontend files.

Before running the repository, make sure to:
